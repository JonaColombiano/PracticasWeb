AJAX

JavaScript Asíncrono + XML (AJAX) no es una tecnología por sí misma, es un término que describe un nuevo modo de utilizar conjuntamente varias tecnologías existentes. Esto incluye: HTML o XHTML, CSS, JavaScript, DOM, XML, XSLT, y lo más importante, el objeto XMLHttpRequest. Cuando estas tecnologías se combinan en un modelo AJAX, es posible lograr aplicaciones web capaces de actualizarse continuamente sin tener que volver a cargar la página completa. Esto crea aplicaciones más rápidas y con mejor respuesta a las acciones del usuario.


Elementos

XMLHttpRequest():Este constructor retorna un objeto XMLHttpRequest por medio del cual podemos comenzar una solicitud y escuchar eventos para controlar el proceso de cominicacion

open(metodo, url, asicrono):el metodo configura la solicitud pendiente. El atributo metódo declar el tipo de metodo HTTP usado para abri la conexion (GET o POST). El atributo url declara la ubicación del código que va a procesar la solicitud. Y asincrono es un valor booleano para declarar si la conexion sera sincrona(false) o asincrona(true). De ser necesario el metódo puede incluir valoeres especificando el nombre de usuario y clave.

send(): este es el metódo que inicia la solicitud. Existen varias versiones de este metodo en un objeto XMLHttpRequest para procesar diferentes tipos de datos. El atributo datos puede ser omitido, declarado como un ArrayBuffer, un blob, un documento, una cadena de texto, o como un FormData

abort(): Este metodo cancela la solicitud


Eventos

loadstrat: Este evento es disparado cuando la solicitud comienza

progress: Este evento se envia periodicamente mienstras se evia o descargan datos

abort: Este evento es disparado cuando la solicitud es abortada

error: Este evento es disparado cuando ocurre un error en la solicitud

load Este envento es disparado cando la solicitud ha sido completada

loadend: Este evento es disparado cuando la solicitud ha sido completada (sin considerar si el proceso fue exitoso o no)
