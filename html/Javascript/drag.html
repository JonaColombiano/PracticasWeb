Drag And Drop

Arrastra un elemento desde un lugar y luego soltarlo en otro es algo que hacemos todo el tiempo en aplicaciones de escritorio, pero ni siquiera imaginamos hacerlo en la web. esto es debido a que las aplicaciones web son diferentes sino porque los desarrolladores nunca contaron con una tecnologia estandar disponible para ofrecer herramientas.

Ahora, gracias a la API Drag and Drop, introducida por las especificaciones HTML 5, finalmente tenemos la oportunidad de crear software que se comportara exactamente como las aplicaciones de escritorio que usamos desde siempre.
