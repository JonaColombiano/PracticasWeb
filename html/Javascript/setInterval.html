setInterval

Ejecuta una función o un fragmento de código de forma repetitiva cada vez que termina el periodo de tiempo determinado. Es decir, cada que pasa determinado tiempo ejecuta la funcion dentro del setInterval en este caso ejecutara el codigo cada segundo

setInterval(funcion que estara ejecutando, tiempo en milisegundos)
