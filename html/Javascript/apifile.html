API File

Nos permite interactuar con archivos locales y procesar su contenido en nuestra aplicacion, la extensión la API File: Directores & System provee las herramientas para trabajar con un pequeño sistema de archivos creado especificamente para cada aplicacion para cada aplicacion, la extensión API File: Write es para escribir contenido dentro de archivos que fueron creados o descargados por la aplicación
